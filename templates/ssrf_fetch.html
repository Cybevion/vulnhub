A misconfigured WAF had an SSRF vulnerability. Attacker:

→ SSRF to 169.254.169.254
→ Got IAM role credentials
→ Listed all S3 buckets
→ Downloaded 100M+ customer records

Cost: $190M in fines + settlements. One SSRF endpoint.
Defence:
• Block 169.254.0.0/16, 10.x, 192.168.x, 172.16-31.x
• Allow only https:// schema
• Validate AFTER DNS resolution
• Use IMDSv2 (requires PUT + session header)
• Isolated network for URL fetching
